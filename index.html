Printi channel utilities

Confused what this is for? Visit channels.printi.me.

rss.js

async function fetchFeed(url)

where
  • url (String) - link to an RSS feed

reddit.js

async function fetchPosts(subreddit = null, sort = null, period = null)

where
  • subreddit (String) - subreddit name, leave empty for all of reddit
  • sort (String) - one of: hot, new, top, rising
  • period (String) - one of: hour, day, week, month, year, all
